Shell 学习

Shell 删除系统日志

第一种：

#!/bin/sh
cd /var/log
cat /dev/null > messages
echo "Logs cleaned up"

缺陷：

  1. 非root用户无法删除；
  2. 未定义变量。

第二种：
带有流程控制及变量的Shell

# 定义日志目录
Log_Dir=/var/log
# 定义root的UID
Root_UID=0

# 判断当前用户是否为root
if [ $UID != $Root_UID ]
then
        echo "You must be root to run the script"
else
        cd $Log_Dir || {
        echo $Log_Dir "is not directory!"
}
fi

# 删除系统日志
if [ -s messages ]
then
        cat /dev/null > messages
        echo "Logs cleaned up!"
else
        echo "Logfile is empty!"
fi
